# 导数题图像可视化

本项目用于展示导数专题题目中的原函数、导函数、零点、驻点与参数变化关系，服务于课堂讲解和学生课后复盘。

公网访问：

https://wcx12.github.io/shuxuepeiyou/领军计划/可视化网页/2026-05-04-导数题图像可视化/

## 讲义关联

当前已在以下讲义目录中写入 `可视化引用.md`：

- `领军计划/讲义/2026-05-04-导数专题-对数指数与多极值点函数最值问题`
- `领军计划/讲义/2026-05-04-导数专题-对数指数与多极值点最值问题`

后续如果讲义新增或重排相关例题，应同步检查 `problems.js` 中的题目顺序、说明文字和讲义引用是否仍然一致。

## 本地预览

在当前目录运行：

```powershell
python -m http.server 8125
```

然后访问：

```text
http://127.0.0.1:8125
```

也可以从仓库根目录启动 `python -m http.server 8080`，再访问 `/领军计划/可视化网页/2026-05-04-导数题图像可视化/`。

## 主要文件

- `index.html`：页面结构和外部依赖入口。
- `styles.css`：页面样式。
- `app.js`：交互逻辑、绘图、零点与驻点检测。
- `problems.js`：题目数据与每题对应的函数定义。

## 依赖说明

页面依赖 `KaTeX` 和 `Plotly` 的 CDN，公网访问时需要联网加载。若后续需要课堂离线演示，应把依赖下载到本项目的 `vendor/` 目录，并同步修改 `index.html` 的资源路径。

## 维护要点

更新题目时，优先修改 `problems.js`，避免把题目数据散落在 `app.js` 中。新增题目后应检查图像区间、参数默认值、零点和驻点标记是否能稳定显示。
