## 维护约定

每个可视化项目单独放在一个目录中，保留 `index.html`、`styles.css`、`app.js`、必要的数据文件和必要的本地依赖。

不要提交 `node_modules/`、本地检查截图、测试结果、临时构建产物或外部平台专用部署脚本。
