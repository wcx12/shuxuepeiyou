# 乘积微分法则可视化

本页面用二维面积和三维体积解释乘积微分法则：

\[
  \mathrm d(uv)=u\,\mathrm dv+v\,\mathrm du,\qquad
  \mathrm d(uvw)=vw\,\mathrm du+uw\,\mathrm dv+uv\,\mathrm dw.
\]

页面通过滑块调整 \(u,v,w,\mathrm du,\mathrm dv,\mathrm dw\)，突出一阶薄片与二阶、三阶高阶小量之间的区别，适合在讲“不定积分前的微分运算回顾”和“微分方程作为导数反推”时使用。

## 本地预览

本页面为单文件静态页面，无外部依赖。可以直接打开 `index.html`，也可以从仓库根目录启动静态服务：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://127.0.0.1:8080/领军计划/可视化网页/2026-05-22-乘积微分法则可视化/
```

修改公式或图形标注后，应同时检查二维模式和三维模式，确认滑块、公式行、解释文字和面积/体积拆分保持一致。
