# 凸函数可视化

本项目配套“凸函数”讲义使用，用交互方式展示凸函数定义、割线斜率、两点 Jensen、中点判别、切线支撑以及若干习题中的核心几何机制。

公网访问：

https://wcx12.github.io/shuxuepeiyou/领军计划/可视化网页/2026-06-20-凸函数可视化/

## 讲义关联

建议配套以下讲义使用：

- `领军计划/讲义/2026-06-20-凸函数`

对应讲义目录中应保留 `可视化引用.md`，说明本页面适用的讲解环节。

## 模块

1. 弦线定义：拖动端点和权重，观察曲线与弦线的高度差。
2. 割线斜率：显示左段、整段、右段三条割线斜率。
3. 两点 Jensen：观察权重如何决定区间中间点。
4. 中点二分：展示二分点如何逼近任意比例。
5. 切线支撑：拖动切点，观察曲线位于切线上方。
6. 最大值上包络：展示两个凸函数的最大值仍为凸函数。
7. 极值两侧单调：展示凸函数极小点左右的单调性。
8. 二阶对称差商：展示局部最大与正二阶差商之间的矛盾。

## 本地预览

本页面为纯静态页面，无外部依赖。可以直接打开 `index.html`，也可以从仓库根目录启动静态服务：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://127.0.0.1:8080/领军计划/可视化网页/2026-06-20-凸函数可视化/
```

## 主要文件

- `index.html`：页面结构。
- `styles.css`：页面样式。
- `scenes.js`：模块名称、说明和控件配置。
- `app.js`：交互逻辑与 SVG 绘图。

## 维护要点

修改讲义内容后，应检查模块顺序、术语和讲义中的命题/练习编号是否仍然一致。页面不依赖 CDN，适合课堂离线演示；如果以后引入第三方绘图库，应优先放入本项目的 `vendor/` 目录并记录来源。
