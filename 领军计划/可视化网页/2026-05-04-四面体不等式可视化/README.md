# 四面体不等式可视化

用于解释四面体不等式问题的交互式静态网页，可观察四面体形状、二面角、面法向量、加权质心、边长、体积与等号模型之间的关系。

公网访问：

https://wcx12.github.io/shuxuepeiyou/领军计划/可视化网页/2026-05-04-四面体不等式可视化/

## 本地预览

直接打开 `index.html`，或在当前目录运行：

```powershell
python -m http.server 8126
```

然后访问：

```text
http://127.0.0.1:8126
```

Three.js 运行时已放在 `vendor/three.min.js`，页面不依赖 CDN。
