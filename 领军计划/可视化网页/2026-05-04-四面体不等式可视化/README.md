# 四面体不等式可视化

本项目用于解释四面体不等式问题的几何结构，可观察四面体形状、二面角、面法向量、加权质心、边长、体积与等号模型之间的关系。

公网访问：

https://wcx12.github.io/shuxuepeiyou/领军计划/可视化网页/2026-05-04-四面体不等式可视化/

## 讲义关联

当前暂无直接对应的讲义目录，因此暂不在讲义中强行添加引用。

后续如果新增四面体不等式相关讲义，应在对应讲义目录中新增 `可视化引用.md`，并写清本页面适用的章节、例题或课堂演示环节。

## 本地预览

直接打开 `index.html`，或在当前目录运行：

```powershell
python -m http.server 8126
```

然后访问：

```text
http://127.0.0.1:8126
```

也可以从仓库根目录启动 `python -m http.server 8080`，再访问 `/领军计划/可视化网页/2026-05-04-四面体不等式可视化/`。

## 主要文件

- `index.html`：页面结构和控件入口。
- `styles.css`：页面样式。
- `app.js`：三维场景、交互和几何量计算。
- `vendor/three.min.js`：Three.js 本地运行时。
- `error.html`：资源加载异常时的辅助页面。

## 依赖说明

Three.js 运行时已放在 `vendor/three.min.js`，页面不依赖 CDN。更新 Three.js 时应记录版本来源，并确认公网页面和本地预览都能正常加载。

## 维护要点

调整几何模型后，应检查边长、体积、法向量方向、二面角和等号模型说明是否仍然一致。若把页面用于具体讲义，应同步补充讲义引用，避免学生无法从讲义定位到对应网页。
